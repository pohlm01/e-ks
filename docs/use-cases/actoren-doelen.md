# Actoren, sub-actoren en doelen

Scope: Eerste Kamerverkiezing

## Actoren en doelen

| Actor                                 | Doel                                                         |
| ------------------------------------- | ------------------------------------------------------------ |
| Politieke groepering                  | Meedoen aan verkiezingen; Verkozen worden                    |
| Ondersteuners (kiezers)               | Politieke groepering op stembiljet krijgen                   |
| Centraal stembureau (CSB)             | Transparante, eerlijke, controleerbare verkiezingen; Correcte, verkiesbare kandidaten op de lijst |
| Kiesraad                              | Transparante, eerlijke, controleerbare verkiezingen; Toegankelijk verkiezingsproces voor alle actoren binnen de wet |
| Ontvangers verkiezingsdefinitie, etc. | Correcte data ontvangen voor vervolg verkiezingsproces       |
| De wereld                             | Geïnformeerd worden en kunnen controleren                    |

Ondersteuners (kiezers) zijn leden van Provinciale Staten en van de Kiescolleges. Per provincie/Kiescollege één nodig.
