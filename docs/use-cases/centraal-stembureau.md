# Centraal Stembureau

## Centraal Stembureau stelt vast dat alle benodigde stukken voor de zittende partij ingeleverd en correct zijn

__Niveau:__ Hoog-over, wolk, ☁️

### Hoofdscenario en uitbreidingen

__Hoofdscenario:__

1. Centraal Stembureau stelt vast dat de kandidatenlijst (model H1) is ingeleverd.
2. Centraal Stembureau stelt vast dat de kandidatenlijst (model H1) correct is. ([LINK] Controles op H1)
3. Centraal Stembureau stelt vast dat de machtingsverklaring (model H3) is ingeleverd.
4. Centraal Stembureau stelt vast dat de machtingsverklaring (model H3) correct is. ([LINK] Controles op H3)
5. Centraal Stembureau stelt vast dat de instemmingsverklaringen (model H9) zijn ingeleverd.
6. Centraal Stembureau stelt vast dat de instemmingsverklaringen (model H9) correct zijn. ([LINK] Controles op H9)


__Uitbreidingen__:

2a. Voorinlevering: Kandidatenlijst is niet correct  
  2a1. Centraal Stembureau vult in het systeem in wat er niet correct is  
    2a1a. Koppelt terug   

2b. Dag KS: Kandidatenlijst is niet correct  
  Centraal Stembureau vult in het systeem in wat er niet correct is  
    2b2a. maakt verzuim aan
